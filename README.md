# mundo-animal

Objetivo:
O objetivo do projeto surgiu com o intuito de acabar com a grande quantidade de animais maltratados e abandonados na rua. O aplicativo tem o objetivo de facilitar as pessoas a adotarem animais, de colocarem para adoção, denunciar, encontrar animais perdidos, ajudar e/ou doar materiais.

Como vai funcionar?
O app vai fornecer suporte para as pessoas que querem doar, adotar ou encontrar um animal que está desaparecido. Ele funcionará por meio de diferentes abas, como adoção/doação, desaparecidos, ajuda especializada (veterinários, adestradores, paseadores, etc), denúncia de maus tratos, doação de materiais (cama, ração, brinquedos…) e uma tela inicial contendo um menu com todas as abas anteriores, o perfil do usuário (configurações), uma área com exposição de eventos gratuitos(castração, vacinação, feiras de adoção, etc) e arrecadação de dinheiro para tratamentos dos animais.
